## Overview

研究型 Agent，从论文 Idea 到完整 Paper 的全流程自动化：

- 💡 **Idea 生成**：从研究方向生成研究问题
- 🔍 **文献调研**：自动搜索、阅读、总结相关论文
- 🧪 **实验设计**：提出实验方案
- 📊 **结果分析**：分析实验结果
- 📝 **论文撰写**：生成完整论文草稿

## Research Pipeline

```
1. Idea Generation
   ↓
2. Literature Review
   ↓
3. Method Design
   ↓
4. Experiment Planning
   ↓
5. Results Analysis
   ↓
6. Paper Writing
```

## Key Features

1. **全流程自动化**：从 Idea 到 Paper
2. **自我进化**：根据反馈迭代改进
3. **文献综述**：自动检索和分析论文
4. **多格式输出**：支持 LaTeX、Markdown
