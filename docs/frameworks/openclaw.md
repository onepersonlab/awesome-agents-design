## Overview

OpenClaw 是一个跨平台的个人 AI 助手运行平台，支持：

- 🖥️ **跨平台**：Windows、macOS、Linux
- 📱 **多渠道**：Telegram、Discord、WhatsApp、Signal 等
- 🧩 **技能系统**：100+ 可扩展技能
- 🏗️ **Manager-Worker 架构**：多 Agent 协作

## Architecture

```
Manager Agent (协调者)
    ├── Task Management
    ├── Worker Orchestration
    └── Human Communication
    │
Worker Agents (执行者)
    ├── Coding Worker
    ├── Research Worker
    └── Content Worker
```

## Key Features

1. **技能系统**：模块化扩展，按需加载
2. **渠道接入**：一键接入主流 IM 平台
3. **记忆系统**：持久化对话记忆 + 向量搜索
4. **子 Agent**：复杂任务自动委托
